quorra.js

subheader

 

quorra.js is an open-source framework for creating rich data-based visualizations, that doesn't require the steep learning curve of other visualization libraries. Out of the box, quorra provides the following features (in addition to plotting):

  • Zooming/Panning
  • PNG Exporting
  • Interactive Plot Annotations
  • Real-Time Updating
  • Layout Tools
  • Data Binding Across Plots

The following plot "models" are currently available within quorra:

Bar Charts

Line Charts

Scatter Charts

Jitter Charts

Step Charts

Heatmaps

Box Charts

Histograms

Density Plots
